# truth-kit

**truth-kit** is a public Kyvero Vexus repository for experimenting with research methods and tools that help people seek truth.

The working thesis is simple: large language models are not truth machines, but they can become useful components in truth-seeking workflows when they are constrained by evidence, provenance, adversarial checking, and reproducibility.

## Goals

- build practical tools for evidence collection and organization
- prototype research workflows that combine human judgment with LLM assistance
- make claims easier to trace back to sources
- improve comparative reading, synthesis, and contradiction detection
- support uncertainty tracking instead of false certainty
- emphasize transparent methods over black-box authority

## Early directions

- source harvesting and normalization
- claim extraction and citation mapping
- evidence graphs and argument structure tools
- contradiction, gap, and inconsistency detection
- provenance-first research notebooks
- reproducible literature review pipelines
- benchmark suites for truth-oriented research tasks

## Principles

1. **Truth over fluency.** A polished answer is not enough.
2. **Sources over vibes.** Claims should point somewhere.
3. **Uncertainty should stay visible.** Confidence is not certainty.
4. **Reproducibility matters.** Others should be able to inspect the path.
5. **Humans remain responsible.** LLMs assist; they do not absolve.
