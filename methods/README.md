# methods

This directory holds the conceptual and procedural foundations of `truth-kit`.

Method documents should define:

- the question type being addressed
- what counts as evidence
- which inference steps are allowed
- how uncertainty should be reported
- known failure modes
- expected output/report structure

Subdirectories:

- `source-criticism/`
- `provenance/`
- `archive-recovery/`
- `authorship-forensics/`
- `uncertainty/`
