# contrib

Low-friction contribution area for external contributors.

If you have useful material but do not want to spend time adapting to full
project layout first, drop it here and open a PR.

Suggested paths:

- `contrib/tools/<name>/` for scripts/tools
- `contrib/writeups/` for specs, notes, RFC-like docs, papers
- `contrib/skills/<name>/` for skill drafts/procedures

Maintainers can relocate and normalize accepted contributions after review.
